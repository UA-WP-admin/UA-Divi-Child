Child theme for Elegant Themes' Divi. These templates are meant to accompany the Division Plugin.


== Description ==

This Divi Child Theme adds addtional action hooks to the header, footer, and sidebar templates. 


== Installation ==

1. Upload 'division.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
